    work flow : 
    github rep -> clone -> changes -> add -> commit -> push

    create folder and create repository in github
    git init : to create a new repository
    git remote add origin [url]t : to add a remote
    git remote -v : to check the remote

    git branch  -
    git branch : to check branch
    git branch -M main : to change the branch name
    git checkout  : to switch to another branch
    git checkout -b  : to create and switch to another branch
    git branch -d  : to delete a branch

    merging branches -
    git diff  : to check the difference between branches
    git merge  : to merge the branch

    undoing changes :
    staged changes :
    git reset filename : to unstage the file
    git reset : to unstage all the files
    commit changes (for one commit):
    git reset HEAD~1 : to undo the last commit
    git log : to check the commit changes
    commit changes(for many commits ) -
    git reset  : to undo the commit
    git reset --hard  : to undo the commit and changes
